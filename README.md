# Висновки щодо правильності розрахунків (Monte Carlo vs аналітика)

Результати моделювання методом Монте-Карло продемонстрували повний збіг із теоретичним розподілом сум при киданні двох кубиків. Емпіричні ймовірності лягли на аналітичну криву, а максимальне відхилення становило лише кілька тисячних, що відповідає очікуваним випадковим флуктуаціям при великій кількості симуляцій.

## Ключові спостереження

форма розподілу коректна — максимум на сумі 7, симетрія відтворена повністю;

емпіричні значення стабільно потрапляють у теоретичні межі;

збільшення кількості кидків зменшує похибку, що підтверджує коректність реалізації та статистичну збіжність.

### Отримані дані підтверджують правильність моделювання, коректність алгоритму та відповідність результатів аналітичній моделі
